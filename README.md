# RAAJJE-THEFT-AUTO-1-THE-BAOKALO-CHRONICLES
RTA v1: The Baokalo Chronicles is a full-featured web-based game built entirely from scratch using only native web technologies (HTML, CSS, JavaScript, Canvas, Web Audio API). It is a top-down action-adventure game set in the Maldives,  The story of Baokalo Ronda, a young man caught between family loyalty, gang life, and political corruption.
