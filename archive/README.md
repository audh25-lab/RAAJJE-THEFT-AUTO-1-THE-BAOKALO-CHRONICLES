# Archive Directory

This directory contains legacy and superseded versions of the Raajje Theft Auto 1 project.

The files here are kept for historical reference and to preserve the development history of the game.

## Contents

- **`legacy/`**: Contains the original HTML5 Canvas 2D prototype files (`game.js`, `index.html`, `graphics.js`, etc.).
- **`webgl_edition/`**: Contains the files from the first Three.js WebGL upgrade (`ultimate-webgl-edition.html`, `webgl-renderer.js`, etc.).

## Current Version

The current, actively developed version of the game is the **Maldivian Edition**.

Please refer to the root directory and the main **`README.md`** for the latest files and documentation.

**Main Entry Point**: `../maldives-edition.html`
**Main Documentation**: `../MALDIVIAN_EDITION_README.md`
